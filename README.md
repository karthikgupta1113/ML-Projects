# Logistic Regression Projects

## 1. Titanic Survival Prediction
- **Dataset:** Kaggle Titanic
- **Goal:** Predict passenger survival

- **Dataset:** Pima Indians Diabetes (Kaggle)
- **Goal:** Predict diabetes in patients

- **Dataset:** Telco Customer Churn (Kaggle)
- **Goal:** Predict which customers will leave

  # Decision Tree Projects

## 1. Iris Flower Classification
- **Dataset:** Sklearn built-in Iris dataset
- **Goal:** Classify flowers into 3 types (Setosa, Versicolor, Virginica)
- **Result:** 100% accuracy at depth 3
- **Key Learning:** Perfect clean dataset, tree visualization

## 2. Heart Disease Prediction
- **Dataset:** Kaggle — Heart Disease Dataset
- **Goal:** Predict whether a patient has heart disease
- **Result:** 85% accuracy at best depth 3
- **Key Learning:** Real overfitting behavior, feature importance (cp, thalach)
